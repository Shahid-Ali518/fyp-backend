# fyp-backend
mental health assistant web platform to facilitate the users about their mental health conditions.
## steps to run the project 
1️⃣ Start Docker Engine

Open Docker Desktop.

On the main screen, if you see Docker Engine: Stopped, click Start or Run (button may say Start Engine).

Wait until it changes to Docker Engine: Running.

This may take a minute or two.

5️⃣ Run Your Project

Now your backend can run:

cd K:\FYP\fyp-backend
docker-compose up --build


FastAPI: http://localhost:8000

MySQL: port 3306
